### Задание №5
Подсчитайте количество заголовочных файлов, файлов с расширением .cpp, сколько остальных файлов (не заголовочных и не .cpp)
```
find . \( -name "*.h" -o -name "*.hpp" \) | wc -l
find . -name "*.cpp" | wc -l
find . -not \( -name "*.h" -o -name "*.hpp" -o -name "*.cpp" \) -type f | wc -l
```
